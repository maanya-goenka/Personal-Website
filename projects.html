Recent Projects

Wanderlust

Swift & Xcode

Developed and designed an application for tourists to access information on places and attractions, track locations, share images with friends and design a customized travel itinerary for themselves. Additional implemented features include a chat feature to enable users of the application to converse with one another, and filters to parse through all the available options and narrow them down to find what you are looking for. Wanderlust is a one stop application for travel planning. This project in its initial stage has been uploaded to github (linked on the top of the website). Updates will be made as changes are made to the current form of the application.

Carlket

Swift, Xcode, Algolia, Firebase, Payment SDKs

Building an iOS e-commerce application with integrated payment system (PayPal) and Stripe SDKs, Firestore and Algolia along with an embedded chat feature. Used Swift, the Xcode playground as well as the Swift storyboard tool, for the programming of the project. This application is aimed at facilitating students at Carleton College to trade or sell used products or goods that they own for financial assistance during the pandemic, sort of like an ebay within the campus premises. This project in its initial stage has been uploaded to github (linked on the top of the website). Updates will be made as changes are made to the current form of the application.
